# Baseline: BETAFPV 900MHz RX

## Device

- Receiver: BETAFPV 900MHz RX
- MCU: ESP32
- RF chip: SX1276
- Current firmware: ExpressLRS RX Firmware v3.3.1
- Region: FCC915

## Goal

Configure local build environment on Lenovo T16 and compile the original/native firmware first.

After the baseline firmware builds successfully, frequency and radio parameters may be modified experimentally.
